# paper-soccer
paper soccer game

### Building:
#### with eclipse
<pre>
Import project to the eclipse:
File -> Import... -> Existing Projects into Workspace
Browse to the downloaded sources, click OK and finish.
</pre>

#### with maven
<pre>
mvn clean package
</pre>

### Running:
<pre>
java -jar target/paper-soccer-1.0.0-SNAPSHOT.jar
</pre>
